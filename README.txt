This is an implementation of a second-order HMM

How to use:

- Checkout code from github (https://github.com/viettrung/trigram-hmm/)
- Go to directory of folder checkout from github
- We have two options to run this tool
Execute the following files with python3:
   - main.py ==> it will find the best tag sequence for a sentence that you entered.
   - test.py ==> it will generate a test result and evaluate the accuracy of this program.

- Wait for result.
- Note: Sentence with new word (not include in Brown Corpus) the result will be wrong --> need care about new word case.
